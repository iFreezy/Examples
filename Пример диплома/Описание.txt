Программа для добавления слов и их описаний в базу данных mssql, для дальнейшего переноса в систему анализа текста для пополнения словника системы.

Программа загружает текстовый файл и далее с помощью регулярного выражения парсит строки, для выделения слов и их описаний. В данный момент это начальная и неотформатированная версия программы, где интерфейс служит, по сути, только для отладки программы.

Интерфейс:
Для удобства просмотра данных в БД имеется модуль Datagridview, где отображается таблица с данными из БД.
Имеется текстовое поле с запросом к БД и также под ним комбобокс для выбора запроса(чтобы не писать в ручную)

Текстбоксы слева и справа отображают данные из файла, отформатированные регулярным выражением и изначальный текст соответственно.
В версии с асинхронными потоками отключены текст боксы, чтобы загружать объемные файлы.
________________________________

База данных локальная, поэтому загрузка в БД без развертывания СУБД MSSQL Server работать не будет(при необходимости могу продемострировать работоспособность программы).
